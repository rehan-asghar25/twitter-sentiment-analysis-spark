# Twitter Sentiment Analysis using Apache Spark

## Project Overview

This project performs sentiment analysis on tweets using Apache Spark MLlib and Logistic Regression.

## Features

- Distributed data processing using Apache Spark
- Text preprocessing and cleaning
- TF-IDF feature extraction
- Unigram and Bigram generation
- Logistic Regression classification
- Accuracy and ROC-AUC evaluation
- Interactive sentiment prediction

## Dataset

Sentiment140 Dataset

## Technologies

- Python
- Apache Spark (PySpark)
- MLlib
- Logistic Regression

## Execution

Run:

python pipeline.py

Benchmark:

python benchmark.py 1
python benchmark.py 2
python benchmark.py 4
